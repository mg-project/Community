# EarthDay
Maogai's homework project

## 项目目标
设计一个世界地球日的介绍网站，参考 https://www.earthday.org/ 的架构和内容，结合大主题“人类命运共同体”和中国特色社会主义理论体系。实现基本的网页跳转和动画功能，在此基础上可以适当拓展。
